# Alvin Chang | 個人網站

張允軒 Alvin Chang 的個人作品集網站，使用純 HTML / CSS / JavaScript 開發，並透過 GitHub Pages 部署。

## 線上預覽
部署完成後可透過以下網址瀏覽：
`https://<你的GitHub帳號>.github.io`

## 特色
- 中英雙語切換（記憶使用者偏好）
- 簡約專業風格、響應式設計
- 單一 HTML 檔案，免框架、免編譯
- 嵌入式 YouTube 影片區塊

## 客製化
- **個人照片**：將 `profile.jpg` 放在同目錄，並將 `index.html` 中相應位置的註解打開
- **YouTube 嵌入**：把 `VIDEO_ID` 換成你要嵌入的影片 ID
- **聯絡資訊**：在 Hero 區塊更新 YouTube / LinkedIn 連結

## 本機預覽
直接用瀏覽器打開 `index.html` 即可。
